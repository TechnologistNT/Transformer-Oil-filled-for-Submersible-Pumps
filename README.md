# СУ · ТМПН · Отпайка v3.2

**Модуль подбора станции управления и расчёта отпайки трансформатора для УЭЦН.**  
Открываешь, вводишь ток и напряжение ПЭД – получаешь готовую рекомендацию по СУ, ТМПН и напряжению на всех ступенях отпаек.

[Открыть модуль](https://technologistNT.github.io/engineering-calculator/tmpn) *(ссылка будет)*

---

## Зачем это нужно

Когда выбираешь станцию управления под новый двигатель, всегда начинается одно и то же:

- Какой ток СУ взять – ближайший больший или с запасом?
- Какой ТМПН поставить, чтобы хватило по мощности и напряжению?
- Какое напряжение выставлять на отпайках для прямого пуска, для частотника, для вентильного привода?

Раньше – листаешь каталоги, считаешь потери в Excel, сверяешься с таблицами. Теперь – просто вбиваешь два числа и смотришь результат.

## Что внутри

| Раздел | Что даёт |
|--------|----------|
| **Расчёт отпаек** | Потери в кабеле, все четыре ступени напряжения (ПП, ЧРП 50 Гц, ЧРП при вашей частоте, ВЭД) с визуальными барами |
| **Подбор СУ** | Автоматический выбор из 40+ моделей (Триол, Электон, Борец, Новомет, Эталон) с ранжированием топ-3 |
| **Рекомендация ТМПН** | Подбор трансформатора по мощности и току с учётом максимального напряжения отпайки |
| **Проверка кабеля** | Допустимый ток с учётом температуры пласта – зелёный/жёлтый/красный статус |
| **Каталог СУ** | Таблица со всеми характеристиками, фильтр по производителю, поиск по модели |
| **Справочник ТМПН** | Мощность, ток, максимальное напряжение – и подбор по вашим цифрам |
| **Памятка ВЭД** | Параметры, пароли и рекомендации для настройки станций разных производителей |
| **Графики** | Характеристика U/f (для ВЭД) и распределение активной/полной мощности/потерь |

Всё в одном окне. Без лишних переключений.

## Как пользоваться

1. Открываешь страницу – сразу видишь схему скважины и поля ввода.
2. Вводишь **ток** и **напряжение** ПЭД (остальные поля – опционально, для точного расчёта).
3. Переключаешь «Быстрый подбор» – если нужен только ток и напряжение, без деталей кабеля и частоты.
4. Смотришь результаты:
   - потери, рекомендуемый ток СУ, мощность, ТМПН;
   - четыре ступени отпаек с цветными барами;
   - топ-3 СУ с рейтингом;
   - проверку кабеля (загрузка в процентах).
5. Переходишь на другие вкладки – каталог, справочник ТМПН, допустимые токи кабеля, режимы СУ, памятка ВЭД.
6. Нажимаешь **«PDF»** – сохраняешь отчёт целиком, со всеми графиками и таблицами.
7. Закрываешь – все значения сохраняются в браузере, открыл – и они уже на месте.

## Почему не Excel

- **Формулы не сломать** – расчёты защищены в коде, ячейки не удалишь.
- **Визуализация** – отпайки, графики, цветовая индикация – всё наглядно.
- **Встроенные базы данных** – каталоги СУ, ТМПН, допустимые токи кабеля – не надо искать в папках.
- **Единый интерфейс** – на компе, планшете, телефоне – одинаково удобно.
- **Тёмная тема** – для работы в поле и ночью.
- **Экспорт в PDF** – готовый отчёт для заказчика или в журнал.

## Технические детали

**Используемые формулы:**

- Потери в кабеле: `ΔU = √3 × I × R_каб`  
  где `R_каб = (удельное сопротивление × L) / 1000`  
  (удельное сопротивление для сечений 10–95 мм² взято из справочных данных).

- Отпайка ПП: `U_пп = U_пэд + ΔU`

- Отпайка ЧРП 50 Гц: `U_50 = U_пп + 30 В`

- Отпайка ЧРП (заданная частота F): `U_f = (U_пп × (F/50)) + 30 В`

- Отпайка ВЭД: `U_вед = U_f × 1.2` (коэффициент зависит от типа двигателя)

**Подбор СУ:**  
Вычисляется требуемый ток `I_су = I_пэд × 1.15`, затем выбирается ближайший больший из стандартного ряда: 63, 100, 160, 250, 400, 630, 800, 1000, 1250, 1600, 2000, 2500, 3000 А.  
Далее выполняется фильтрация каталога по производителю, типу двигателя (АД/ВЭД) и минимальному току. Ранжирование – по приоритету производителя (Электон → Эталон → Борец → остальные) и току.

**Подбор ТМПН:**  
Перебираются все модели (63–1000 кВА), выбирается первая, у которой мощность ≥ расчётной, ток ≥ требуемого, а максимальное напряжение (Uвн макс) ≥ расчётной отпайки ВЭД. Если такой нет – выдаётся предупреждение.

**Коррекция допустимого тока кабеля:**  
Температурный коэффициент: 1.0 при ≤25°C, 0.91 при 35°C, 0.82 при 45°C, 0.71 при 55°C, 0.58 при >55°C.

## Источники данных

- **Каталог СУ** – собраны открытые данные производителей: Триол (серии АК06 RC/LH), Электон (05, 07), Борец (15, ПП), Новомет (СУ-04, 05), Эталон (СУ ЧР АВ).  
- **ТМПН** – стандартный ряд мощностей и токов по ГОСТ.  
- **Допустимые токи кабеля** – по ПУЭ с поправкой на температуру.  
- **Памятка ВЭД** – составлена на основе реальных инструкций по настройке станций и опыта эксплуатации.
